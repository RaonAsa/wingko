이클립스 Tool

STS : Spring Tool Suite
download : http://spring.io/tools/sts/all

1. 사용하는 o/s 버전 zip파일 다운로드

2. 압축경로\springsource\sts-3.3.0.RELEASE\STS.exe 실행
 - 개발시 사용한 버전 : sts 3.3버전

3. STS 적용된 이클립스 실행되면 프로젝트 소스 import
